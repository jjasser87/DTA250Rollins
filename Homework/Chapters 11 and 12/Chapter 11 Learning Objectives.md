1.	Understand Basic String Manipulation in R:
–	Identify and use the basic string manipulation functions provided by the base R package.
–	Count the number of characters in a string using nchar().
–	Extract substrings from a string using substr().
–	Concatenate strings using paste().
–	Convert strings to uppercase or lowercase using toupper() and tolower().
–	Remove leading and trailing whitespace from strings using trimws().
2.	Utilize Advanced String Manipulation Functions with the stringr Package:
–	Load and utilize the stringr package for more powerful string manipulation.
–	Count characters, extract substrings, and concatenate strings using str_length(), str_sub(), and str_c().
–	Convert strings to uppercase or lowercase using str_to_upper() and str_to_lower().
–	Remove leading and trailing whitespace using str_trim().
–	Replace substrings within a string using str_replace().
–	Detect patterns within strings using str_detect() and count their occurrences using str_count().
–	Extract specific patterns using str_extract() and str_extract_all().
3.	Apply Regular Expressions for Pattern Matching:
–	Understand the basics of regular expressions and their application in string manipulation.
–	Use regular expressions with functions like grep() and grepl() to search for patterns in text data.
–	Apply regular expressions for advanced text processing tasks such as identifying specific text patterns.
4.	Perform Practical String Manipulation Tasks:
–	Clean and preprocess textual data by removing punctuation and unnecessary whitespace.
–	Conduct simple sentiment analysis on textual data using string manipulation functions.
–	Visualize the results of text analysis using the ggplot2 package.
5.	Analyze Text Data Using String Manipulation:
–	Load and manipulate the IMDB review dataset to perform sentiment analysis.
–	Create new variables in the dataset based on the presence of positive or negative sentiment words.
–	Generate visualizations to display the results of the sentiment analysis.
